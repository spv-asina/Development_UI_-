# ТЗ:
Робот: РоботХимик

Описание:
РоботХимик выполняет задачи в контексте 'Химии'.
Координатная система: ось Y направлена вверх (на север), ось X направлена вправо (на восток). Начало координат — в нижней левой ячейке.

Файлы диаграмм:
- robot.png - диаграмма классов робота для реализации и сопоставления
- dir1.png и dir2.png — диаграммы соответствия методов движения и направлений
- iterator.png — модель итератора (зигзаг: вправо, вверх, влево, вверх, вправо...)

Методы специализации и их действие (явно):
- Реактив: заменяет Реактив → Обработано
- Пусто: заменяет Пусто → Реактив

Цель:
- После завершения работы не должно остаться ячеек следующих типов:
  - Реактив
  - Пусто
- Программа должна завершиться на ячейке типа: Финиш

Ограничения перемещения:
- нельзя заходить на ячейки типа: Опасно
- нельзя заходить на ячейки типа: Барьер

В коде использовать только enum.value при передаче в функции. Пример: ИмяПеречисления.Опция.value

После завершения работы вам нужно создать файл mapping.yaml следующего вида:
классы:
   - Класс диаграммы: ВашКлассВПитоне
     свойства:
        - свойство_класса_в_диаграмме: свойство_класса_в_коде
     методы:
        - метод_класса_в_диаграмме: метод_класса_в_коде
перечисления:
   - ПеречислениеНаДиаграмме: ПеречислениеВКоде
     опции:
        - опция_на_диаграмме: опция_в_коде
Весь необходимый код должен быть в файлу main.py. Необходимо отправить mapping.yaml и main.py на проверку преподавателю.
